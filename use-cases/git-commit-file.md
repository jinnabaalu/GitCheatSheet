# Committing files

## Status

```bash
git status
```

## Git Add

```bash
git add .

# OR

git add --all

# OR

git add hello-world.js

# OR

git add dir/
```

## Check status

```bash
git status
```

## Commit

```bash
git commit -am "hello world changes"
```

`-am` : All commits with message

## Push

```bash
git push -u origin master

# OR

git push # pushes the current branch to the configured upstream
```
